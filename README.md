# Professional Portfolio Website

A high-performance, responsive, and premium-designed personal portfolio website built with React and Vite.

## Features

- **Modern & Premium Design:** Dark mode aesthetic with custom color palette and glassmorphism effects.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop.
- **Sections:**
  - **Hero:** Impactful introduction with call-to-action.
  - **About:** Professional bio and statistics.
  - **Skills:** Categorized technical expertise.
  - **Projects:** Showcase of work with tech stack details.
  - **Services:** Clear offering of professional services.
  - **Contact:** Functional form layout and contact details.
- **Tech Stack:** React, Vanilla CSS (CSS Modules approach), Vite.
- **SEO Optimized:** Meta tags and semantic HTML structure.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Customization

- Update `src/components` with your personal content.
- Modify `src/index.css` to change the color scheme (CSS Variables).
